## How to Add Topics

1. Go to https://github.com/StableExo/TheWarden
2. Click "⚙️" (Settings) icon on the right sidebar
3. Click "Topics" input field
4. Add topics one by one (GitHub will suggest existing ones)
5. Press Enter after each topic
6. Topics appear as clickable badges under repository name
